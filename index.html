Plano Acelerado — 20 Semanas

Eixo principal: Curso iOS Developer (Tiago Aguiar). Curso de Swift = suporte sob demanda. Cada semana produz algo funcional.

0

Ignição

Semanas 1–2

Curso Tiago Aguiar Módulos 1-4 + Lógica em Swift + Git. Construir 2 mini-apps pessoais.

Recursos:

  • iOS Developer — Tiago Aguiar (Módulos 1-4)Eixo principal

    Preparação + Ambiente + Fundação iOS + Apple SDK.

  • Curso de Swift — Tiago AguiarSob demanda

    Consultar quando precisar. NÃO seguir linearmente.

Entregável: 2 mini-apps pessoais no GitHub (MoodTracker + Decisor) + 14 commits
1

Primeiro App Real

Semanas 3–6

Curso Tiago Módulos 5-7 (SwiftUI → APIs). Zero C — imersão total Swift. Publicar Projeto 1.

Recursos:

  • iOS Developer — Tiago Aguiar (Módulos 5-7)Eixo principal

    SwiftUI intermediário + Apps Online (URLSession, JSON, Codable, Combine).

Entregável: Projeto 1 "AcademyPrep" publicado (3+ telas, persistência, README)
2

Evolução

Semanas 7–11

Curso Tiago Módulos 8-10 (Combine, Storage, Cupertino). C e Raciocínio Lógico em paralelo. Ideação CBL na sem 11.

Recursos:

  • iOS Developer — Tiago Aguiar (Módulos 8-10)Eixo principal

    Combine, LocalStorage, UI Cupertino, Design Patterns.

  • Matemática Rio — Raciocínio Lógico30 min/dia

    Sequências, proposições, diagramas. Em paralelo.

Entregável: Projeto 2 publicado + problema CBL definido + simulado lógica 70%+
3

Projeto Principal

Semanas 12–16

O projeto que te diferencia. CBL aplicado de verdade, impacto real, design intencional.

Recursos:

  • Apple — Challenge Based Learning GuideEssencial

    Big Idea → Essential Question → Challenge → Solution.

  • Apple — Human Interface GuidelinesEssencial

    Design guidelines oficiais. Diferencial real na Academy.

Entregável: Projeto principal com CBL + README + vídeo demo + pitch de 3 min
4

Polimento Final

Semanas 17–20

Portfolio impecável, simulados, preparação completa. Chegar pronto.

Recursos:

  • Playlist "Estudos Apple Developer Academy | UCB" (YouTube)Essencial

    Dicas de ex-alunos e resoluções de provas anteriores.

Entregável: Portfolio GitHub completo, simulado 80%+, pitch afiado, entrevista simulada
Regra de ouro: Curso do Tiago = eixo. Curso de Swift = dicionário. Commit diário. Todo projeto resolve um problema real.
