Plano Acelerado — 20 Semanas

Princípio: Construir > Estudar. Swift desde o dia 1. Cada semana produz algo concreto.

0

Ignição — Swift + Git

Semanas 1–2

Começa a programar em Swift HOJE. Zero teoria pura. Git desde o dia 1.

Recursos:

  • Swift Playgrounds (app gratuito da Apple) Melhor opção

    Aprenda Swift de forma interativa no iPad ou Mac.

  • Curso em Vídeo — Git e GitHub Essencial

    20 aulas. Git é obrigatório desde o dia 1.

Entregável: 2 mini-apps no GitHub + 14 commits (1/dia)
1

Fundação Swift

Semanas 3–6

Dominar SwiftUI básico e publicar o primeiro projeto real. C começa em paralelo (só o essencial para prova).

Recursos:

  • 100 Days of SwiftUI (Hacking with Swift) Melhor opção

    Paul Hudson. Projeto-a-projeto. Dias 1–25 cobrem o essencial.

  • Curso em Vídeo — Linguagem C Para prova

    Só os fundamentos: variáveis, arrays, funções. Não precisa aprofundar.

Entregável: Projeto 1 publicado no GitHub (3+ telas, persistência, README)
2

Construção

Semanas 7–11

SwiftUI intermediário, OO em Swift (não Java), APIs, persistência avançada. C para prova.

Recursos:

  • 100 Days of SwiftUI (continuação) Melhor opção

    Dias 25–50: MVVM, APIs, Codable, Core Data.

  • Matemática Rio — Raciocínio Lógico Para prova

    Sequências, proposições, diagramas. 30 min/dia em paralelo.

Entregável: Projeto 2 publicado (4+ telas, API, MVVM, acessibilidade) + simulado lógica 70%+
3

Projeto Principal

Semanas 12–16

O projeto que te diferencia. CBL aplicado de verdade, impacto real, design intencional.

Recursos:

  • Apple — Challenge Based Learning Guide Essencial

    Metodologia oficial. Big Idea → Essential Question → Challenge → Solution.

  • Apple — Human Interface Guidelines Essencial

    Design guidelines oficiais. Diferencial real na Academy.

Entregável: Projeto principal com CBL + README + vídeo demo + pitch de 3 min
4

Polimento Final

Semanas 17–20

Portfolio impecável, simulados, preparação completa. Chegar pronto.

Recursos:

  • Playlist "Estudos Apple Developer Academy | UCB" (YouTube) Essencial

    Dicas de ex-alunos e resoluções de provas anteriores.

Entregável: Portfolio GitHub completo, simulado 80%+, pitch afiado, entrevista simulada
Regra de ouro: Projeto > Tutorial. Commit diário. C é meio, não fim. Swift é o core. README é seu pitch escrito.
